 1. Perceive 
 2. Intercept (and Update my Knowledge of my world => Memory Model (short or Long ?)
 3. Plan STEPS => Reason => Decide
 4. Act accordingly
 5. Learn and maybe update my knowledge
 6. Loopback to step 1 if time or loop limit is not reached
